# Neural-Net-with-Numpy
# NumPy Neural Network – From Scratch

A lightweight, educational neural network implementation built **entirely with NumPy**, demonstrating the core mechanics of feedforward computation, backpropagation, and gradient descent — without using any deep learning frameworks (PyTorch, TensorFlow, Keras, etc.).

This project is ideal for students, self-learners, and anyone who wants to deeply understand how neural networks actually work under the hood.

## Features

- Fully hand-written forward and backward passes
- Supports multiple hidden layers
- Configurable activation functions (ReLU, sigmoid, tanh)
- Mean Squared Error (MSE) and Binary Cross-Entropy loss
- Mini-batch gradient descent with momentum (optional)
- Simple but clean object-oriented design
- Detailed vectorized NumPy operations (no explicit Python loops where avoidable)
- Training progress visualization (loss curve)
